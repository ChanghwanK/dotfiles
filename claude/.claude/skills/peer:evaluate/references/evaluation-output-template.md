# 평가 리포트 출력 형식

아래 형식을 그대로 사용하여 최종 리포트를 출력한다.
`{...}` 로 표시된 부분을 실제 내용으로 채운다.

---

**분석 기준일**: {오늘 날짜}

## 작성 시 주의사항

1. **STAR 구조 완결성**: Situation, Task, Action, Result 4개 항목을 모두 채운다. 증거가 부족하면 해당 항목을 생략하기보다 "기록 미확인"으로 명시한다.

2. **점수와 서술 일관성**: 점수 4점이면 Yes 질문이 70%+ 이어야 한다. 서술 내용과 점수가 불일치하지 않도록 점수를 마지막에 확정한다.

3. **기대 사항 톤**: "~하지 않았습니다"(비판)가 아닌 "~하면 좋겠습니다"(기대) 형태를 사용한다.

4. **정량적 결과 우선**: Result 항목에 시간, 비용, 건수 등 수치가 있으면 반드시 포함한다.

5. **증거 건수 명시**: 푸터에 실제로 분석한 observation 건수를 정확히 기입한다.
